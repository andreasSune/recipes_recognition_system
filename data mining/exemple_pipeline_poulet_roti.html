Graphe de la recette : Poulet rôti
La recette est stockée dans SQLite comme un graphe dirigé. Chaque nœud est un geste physique observable, chaque flèche est une transition.
rincer → assaisonner → badigeonner → enfourner → arroser → découper
Parmi +1 million de recettes dans la base · Type : FOUR · Famille : POULET
FSM — Patterns fréquents détectés dans le graphe
Le FSM parcourt le graphe et identifie les sous-séquences qui reviennent dans au moins 5% des recettes du corpus.
Patterns trouvés dans le poulet rôti :
assaisonner → enfourner support 34%
badigeonner → enfourner support 28%
enfourner → arroser support 21%
rincer → assaisonner support 8%
Ces patterns deviennent les features initiales du vecteur binaire qui représente la recette pour le GNN.
Vecteur binaire résultant
[ 1, 1, 1, 0, 1, 0, 0, 1, 0, ... ]
1 = pattern présent dans la recette · 0 = absent
MBA · Apriori — Co-occurrences de gestes
Quels gestes du poulet rôti apparaissent souvent ensemble dans le corpus ?
{badigeonner, assaisonner} → {enfourner} · lift 3.2
{rincer} → {assaisonner} · lift 2.8
PrefixSpan — Gestes déclencheurs identifiés
Si le capteur observe badigeonner comme premier geste, quelle est la suite la plus probable ?
badigeonner → enfourner → arroser
badigeonner est un geste déclencheur fort pour les recettes de type FOUR · POULET (lift 4.1). Dès qu'il est observé, 80% des recettes non-volailles sont éliminées immédiatement.
Clustering — À quel groupe appartient le poulet rôti ?
K-Means appliqué sur les vecteurs FSM regroupe les recettes en familles naturelles. Le poulet rôti tombe dans :
Cluster 7 — Viandes au four
82% de similarité avec le centre du cluster
Recettes voisines dans le même cluster :
agneau rôti poulet en cocotte canard à l'orange rôti de porc
Le GNN n'aura plus à distinguer 1M de recettes — seulement les ~400 recettes du cluster 7. Espace réduit d'un facteur 2500.
GNN — Apprentissage de l'embedding du poulet rôti
Le GNN reçoit le graphe de la recette. Chaque nœud passe par 2 couches de message passing.
Couche 1 — chaque geste intègre ses voisins directs :
assaisonner apprend que ses voisins sont rincer et badigeonner
Couche 2 — contexte élargi :
assaisonner sait maintenant qu'il mène à enfourner puis arroser
Readout — vecteur final du graphe :
embedding = [ 0.82, -0.31, 0.55, 0.19, -0.74, ... ]
vecteur de 128 dimensions représentant toute la recette
Les recettes du cluster 7 partageant les mêmes patterns FSM sont proches dans l'espace latent — c'est la validation sans étiquettes.
